# Hospital Wait Forecast

Forecasting hospital waiting times from operational data.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m hospital_wait_forecast
```

## Project structure

```
hospital_wait_forecast/        # source package
tests/      # tests
requirements.txt
```
